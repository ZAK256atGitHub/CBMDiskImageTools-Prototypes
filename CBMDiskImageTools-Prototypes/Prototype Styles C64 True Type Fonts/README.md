## Prototype Styles C64 True Type Fonts
Das Projekt zeigt die Verwendung der TrueType Schriftarten "C64 Pro Mono" und "C64 Pro" von style64.org.
Das Projekt verwendet die Schriftarten "C64 Pro Mono" und "C64 Pro" ( https://style64.org/c64-truetype ), welche installiert sein müssen.
